开始阅读介绍

简洁优先

以 Markdown 为中心的项目结构，帮助您专注于写作。

Vue 驱动

享受 Vue 的开发体验，在 Markdown 中使用 Vue 组件，并使用 Vue 开发自定义主题。

高性能

VuePress 为每个页面预渲染静态 HTML，页面加载后作为 SPA 运行。

主题

提供开箱即用的默认主题。您也可以选择社区主题或创建自己的主题。

插件

灵活的插件 API，允许插件为您的网站提供许多即插即用的功能。

构建工具

默认构建工具是 Vite，同时也支持 Webpack。选择您喜欢的那个！
